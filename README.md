# Sci-Fi Empfehlungen — Android App

Hendriks Sci-Fi Buchempfehlungen als installierbare Android-App

**143 Titel** mit Cover-Bildern, Reading Tracker, Export und Filtern.

Lädt die Inhalte live von: https://hendr15k.github.io/sci-fi-empfehlungen/

## Features
- 📚 143 kuratierte Sci-Fi-Buchempfehlungen
- 🖼️ 131 Cover-Bilder (OpenLibrary)
- 📖 Reading Tracker ("Habe ich gelesen")
- 📊 Export CSV/JSON
- 🌙 Dark/Light Mode
- 🔍 Volltextsuche + Filter + Sortierung
- 🎲 Random-Picker
